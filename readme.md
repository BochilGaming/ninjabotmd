## INSTALASI DI VPS & RDP

```bash
git clone https://github.com/amiruldev20/ninjabotmd

cd ninjabotmd

npm i


-- jika ts-node dan nodemon tidak terinstall, silahkan install manual dengan command dibawah
npm i -g ts-node
npm i -g nodemon
```
* perintah jalankan: npm run dev
